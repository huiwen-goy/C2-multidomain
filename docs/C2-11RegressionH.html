Comparison with Hamalainen et al.results

back to top

Similar findings:
* Worse PTA, higher asymmetry, self-report more trouble hearing in noise ~ more hearing difficulty/subjective HL
* Memory (higher PC scores for their sample, subjective worsening memory for our sample), driving/good transport, higher social network index ~ more hearing difficulty/subjective HL

They found these effects, but we did not:
* Age, male sex, device use, higher multimorbidity


Variance inflation factor (VIF)

back to top

To check for multicollinearity, VIF is calculated for each predictor as 1 / (1 - R-squared), where R-squared is calculated from regressing all other predictors on that one predictor. A VIF of 5 means that 0.8 of the variance in that predictor is explained by all the other predictors. All the VIF values are well under 5, so multicollinearity is not an issue with these 24 predictors.

High-leverage datapoints

back to top

High-leverage datapoints have very different values from other datapoints, and therefore have more potential to influence the regression line. A leverage statistic h can be calculated for each datapoint, which takes into account multiple predictors for that datapoint. h values range between 1/n and 1 (n = sample size), and values over 2p/n (p = number of predictors) are considered high-leverage. In this case, 50/365 datapoints have h values greater than 0.13.
